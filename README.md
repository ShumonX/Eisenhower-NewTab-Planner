# Eisenhower Planner

Replaces the New Tab page with an Eisenhower Matrix.

No account is needed, no data is collected.

To ensure you are not overwhelmed and that you finish all your tasks for the day, try to put less than 6 tasks in total.

Press the "Clear All" button at the end of every evening.

And lastly — write the tasks for next day on the PREVIOUS evening.

You can check Eisenhower method on Wikipedia, or Eisenhower's Matrix on any productivity blog on the internet!
